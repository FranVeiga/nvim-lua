# FranVeiga's Nvim-Lua

This is my Neovim config written in Lua. It takes a lot of configuration ideas and code from [Lunarvim](lunarvim.org).

## Plugins

My config includes:

- Treesitter
- lspconfig
- lsp-installer
- nvim-cmp
- barbar
- nvim-commenter
- vim-surround
- Telescope
